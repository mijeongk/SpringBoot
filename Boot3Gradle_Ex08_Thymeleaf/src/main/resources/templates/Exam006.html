VO 사용하기

객체 출력

[[${person}]]

VO 개별 접근하기

이름 : [[${person.name}]]
나이 : [[${person.age}]]
성별 : [[${person.gender}]]
성별 : [[${person.gender ? "남자" : "여자"}]]
성별 : [[${person.gender==false ? "남자" : "여자"}]]
성별 : [[${!person.gender ? "남자" : "여자"}]]
성별 : 남자 여자

.대신 []로 사용하여 접근

이름 : [[${person['name']}]]
나이 : [[${person['age']}]]
성별 : [[${person['gender'] ? "남자" : "여자"}]]

Getter를 사용하여 접근

이름 : [[${person.getName()}]]
나이 : [[${person.getAge()}]]
성별 : [[${person.isGender() ? "남자" : "여자"}]]

블록으로 잡고 객체명 생략(*{}사용)

[[*{name}]] ([[*{age}]]세, [[*{gender ? "남자" : "여자"}]])
